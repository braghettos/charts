braghettos/charts

Classic Helm registry for Krateo charts — two channels, one GitHub Pages site.
OCI (oci://ghcr.io/braghettos/charts/…) remains the source of truth.

blueprints

User-installable Krateo marketplace blueprints (schema + CompositionDefinition + tile metadata).

helm repo add braghettos-blueprints https://braghettos.github.io/charts/blueprints
helm repo update
helm search repo braghettos-blueprints

Index: blueprints/index.yaml

operators

Operator/KOG install charts, CRD/helper/target subcharts, and plain dependency/app charts. Installable via helm; not rendered as marketplace tiles.

helm repo add braghettos-operators https://braghettos.github.io/charts/operators
helm repo update
helm search repo braghettos-operators

Index: operators/index.yaml

Chart tarballs are served from this repository's GitHub Releases. See the repository for the publishing contract.
